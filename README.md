Practico2 es un proyecto en Python que simula el funcionamiento básico de una cuenta bancaria. 
Permite crear una cuenta con su número asociado y realizar operaciones como ingresar dinero, retirar, consultar saldo y visualizar movimientos.
El objetivo del proyecto es practicar lógica y conceptos fundamentales de programación.

Al ejecutar el programa, se mostrará un menú que permite realizar operaciones como:

Ingresar dinero

Retirar dinero

Ver el saldo

Ver los movimientos realizados

Objetivo del Proyecto

Este proyecto fue desarrollado como práctica para reforzar:

Programación orientada a objetos (si la usas)

Manejo de funciones

Listas y estructuras de datos simples

Validaciones

Flujo lógico de un programa interactivo
